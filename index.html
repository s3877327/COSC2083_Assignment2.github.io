Overall, I found the feedback to be constructive and is useful both for my current project and for writing my future reports. Their feedbacks give me insights and tips to modify my report. The most important thing from their feedback is that they make me realize that is I want to compete with the classics of old, graphic and technology alone isn’t enough. Although it has been a clear objective since the initial ideas for this project came up in my mind that I want to stay faithful to the past’s establishment, to honor the bricks that help define the modern gaming industry, but with the implementation of contemporary philosophy and technology, such as the implementation of AI and machine learning. One of my friend’s feedback helps me realize that if I make a game in that regard then although I can stay true to my pre-defined objective, almost nothing can help my project stand out from the classics. Which makes the project’s finished product becomes nothing more than an imitation of the past’s achievement. Besides that, for future projects, according to the feedbacks, I should include more sample images from the project to better visualize and to imagine the contents. For this project’s future – that is, if this project is chosen to implement for assignment 3, It is too late to change the project’s objective, so it is more suitable to implement extra elements and contents that separate itself from the project’s main gameplay. If possible, the project should have an extra mode that implements the applications of AR technology, where the game instead of taking place in a pixel galaxy far away, will play out with real life’s sky as background, no matter if it is a clear day or a night sky. Couple that with the explosions and destructions of buildings, should any appear in the background, this will be a very nice side gig that not only preserves the project’s objective by not alter the core gameplay but also separates this project from the classics of the old.
